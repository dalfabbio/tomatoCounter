THE MARVELLOUS TOMATO COUNTER

Are you familiar with the Pomodoro Tecnique?

  1. Pick a task
  2. Focus for 25 minutes
  3. Draw a tomato and take a short break (like 5 minutes)
  4. Focus again for 25 minutes
  5. Every four tomatoes, take a longer break (15/30 minutes)

At the end of the day check how many tomatoes you have!

You can use the tools below to:

  • Keep count of your tomatoes with the counter
  • Set a number of tomatoes to start with
  • Set up a stopwatch that every 25 minutes draws a tomato
